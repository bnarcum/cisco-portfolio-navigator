Constellation (today)

Force-directed 3D graph. Good for seeing global mesh density and integration links. Nodes drift; layout is emergent, not stable geography.

Gallery layout (proposed)

Curated 3D placement — mesh3d-style floating panels. Same LINKS data; position by pillar zone + link clustering. Products expand in spatial orbit on focus (like Composition, but 3D).

What this mockup is for

  • Alternative Spatial layout mode — not a sixth tab, not editorial stats
  • Family ↔ family: platform zones + link beams (direct / adjacent / succession)
  • Family → products: orbital shelf when selected (reuses composition data)
  • Real build: Three.js planes at fixed coordinates from pillar layout + link force only for fine-tuning
  • mesh3d borrow: visual polish (panels, depth, camera) — not their “state of the gallery” content model
